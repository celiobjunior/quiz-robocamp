# Estrutura do Projeto

## Pastas principais
- `src/`: aplicação React (páginas, componentes, contexto, dados e estilos).
- `assets/`: arquivos auxiliares do projeto.
- `docs/`: documentos de referência e apoio.

## Organização de arquivos antes soltos na raiz
- Prompt de design movido para: `docs/design/prompt.md`.
- Logos originais (fonte) movidos para: `assets/source-logos/`.

## Observação
A aplicação usa os logos em `src/assets/brand/`.
